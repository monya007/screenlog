INTRODUCTION
------------
This module provides integration for the screenlog.js plugin. screenlog.js 
brings the developer console right on your screen so that you can do logging 
without having the developer console open always.


REQUIREMENTS
------------
The Screenlog module requires the Libraries API module.


INSTALLATION
------------
1. Copy record question folder to modules directory (usually sites/all/modules).
2. Download the library https://github.com/chinchang/screenlog.js/zipball/master
3. Insert the contents of the archive here
sites/all/libraries/screenlog.js
4. At admin/build/modules enable the Stacktable module in the Development package.
5. At admin/config/development/screenlog specify the desired paths and check checkox to enable screenlog.

For detailed instructions on installing contributed modules see:
http://drupal.org/documentation/install/modules-themes/modules-7
